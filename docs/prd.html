今天換誰？

Task Marketplace + Reward System

「今天換誰？」不是一個家事管理 App。它是一個讓家人、情侶、室友透過獎勵發起任務、主動接單，並一起完成生活大小事的任務平台。

版本：MVP v1.0 文件：Product Requirements Document 日期：2026-07-11

1產品定位

✕ 我們不是
  • 家事管理 App
  • 待辦清單（Todo List）
  • 排班／輪值表（Flatastic 已在做）
✓ 我們是
  • Task Marketplace（任務市集）
  • Reward System（獎勵系統）
  • 讓每個人「主動」接下任務的平台

核心理念

不是「今天輪到哥哥」，而是「今天誰想拿？👀」
—— 用獎勵驅動主動接單，取代排程與輪流的被動指派。

目標對象

家人 👨‍👩‍👧‍👦、情侶 💑、室友 🏠、小型團隊（如公司行政）—— 任何共同生活或共事、需要分攤生活大小事的小群體。

品牌即產品

「今天換誰？」本身就是一句自然的日常對話，也是一個很好的品牌：

  • 👀 今天換誰洗碗？
  • 👀 今天換誰倒垃圾？
  • 👀 今天換誰遛狗？

星星只加不扣 —— 永遠不能扣。這是產品的核心原則。

  • App 內通知中心（未讀列表）
  • 群組層級開關（可靜音某個群組）

3MVP 核心流程

建立群組→ 建立任務→ 任務出現在任務看板→ 有人接單→ 完成一次→ 等待確認→ +1 ⭐→ 完成→ 領取獎勵→ 結束

每個關鍵節點（新任務、接單、待確認、+1⭐、完成）都會觸發 F6 的對應通知。

色彩比例建議
