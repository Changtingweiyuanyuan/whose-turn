# 今天換誰？設計語言（v4 — 黑底顆粒雜誌刊物風）

參考基準：使用者核可的黑底雜誌版面（WHOSE TURN TODAY 刊頭 + 顆粒背景 + 藍/白輪替卡）。

## 核心原則

1. **雜誌刊物感**：像一本每週出刊的家事週刊，不像 SaaS App。
2. **黑底顆粒為主**：全站底色 #010101 + CustomPainter 雜訊顆粒（`NoiseBackground`，套在外殼 `home_shell`，所有分頁共用）；文字白色；卡片是彩色色塊浮在黑底上。
3. **大字刊頭**：每頁上方都是雜誌刊頭（見下）。
4. **彩色色塊卡**：卡片實色填滿（藍灰 `main` / 白輪替）、圓角 8、無邊框無陰影。
5. **重點色克制**：粉 `pink` 是主 accent（誰、選中、進行中、建立鍵、底線、NO.xx）；橘 `orange` 用於已完成/次數強調。

## 色票（唯一來源：`lib/theme/app_colors.dart`）

| Token | Hex | 用途 |
|---|---|---|
| ink | #010101 | 全站底色、主要按鈕、卡片深色文字 |
| diluteInk | #222222 | 底部導覽列底、獎勵標籤邊框 |
| main | #C2D1D3 | 藍灰卡片底、未選中分頁 |
| white | #FFFFFF | 白卡片底、深底上文字 |
| pink | #CF729B | accent：誰／選中／進行中／建立鍵／NO.xx |
| orange | #FF8B04 | 已完成、次數強調 |

## 字重（只用這 4 階，不得新增其他）

| 字重 | 用途 |
|---|---|
| **w900** | 首頁大標「今天換誰？」（唯一） |
| **w800** | 頁標題、卡片標題、次數、狀態標籤、NO.xx |
| **w600** | 刊頭 kicker、分頁、次要標題、獎勵文字 |
| **w500** | 內文最細、輔助說明 |

> 禁止使用 w100–w400、w700。新增文字一律從這 4 階挑。

## 間距 / 圓角（token：`lib/theme/app_tokens.dart`）

- **圓角**：按鈕/輸入框 = 6（theme 全域 `radius`）；卡片 = 8（cardTheme / `AppRadius.card`）；獎勵標籤 = 膠囊（999）。
- **任務卡內間距**（`task_card.dart`）：圖片↔內容 12、標題↔發起人 4、發起人↔獎勵 8、獎勵↔動作 8。
- **分頁**：文字↔底線 4px。

## 元件規範

- **刊頭 Masthead**：`WHOSE TURN TODAY`（白 w600、字距 3、單行）+ `NO.xx`（粉 w800，xx = 目前使用者在群組的序號，補 0 兩位數）→ 粉色 2px 分隔線 → 大標。
  - 首頁：大標「今天換誰？」64px w900，「誰」粉色。
  - 我的任務：大標「我的任務」40px w800，副標「已累積 ⭐ N」。
- **分頁 Tabs（滑動底線式）**：白字，選中 w800 + 粉色底線；未選中 w600 + 灰。切換時底線以 AnimatedAlign / TweenAnimationBuilder（240ms easeOutCubic）滑動。首頁靠左排列；我的任務三格平均分佈 + 整條基線。
- **任務卡**：藍灰/白輪替、圓角 8、無框無影。左大 emoji、標題大字、發起人小灰字（冒號 w800）、粉色獎勵藥丸；次數右上（大數字 + 小 /N）；動作右下（我要接 = 墨黑按鈕帶 → / 進行中 = 粉色小字）。
- **獎勵標籤**：白底膠囊 + `diluteInk` 1.5px 邊框 + 墨黑文字（純文字、無 icon）。
- **底部導覽**：`diluteInk` 底、圓弧 notch、`extendBody: true`（缺口露黑底）；中央建立鍵 = 粉色圓形；選中項粉色、未選中 white54。
- **背景顆粒**：`NoiseBackground`（opacity 0.2 / density 0.1，可調）。

## 修改 UI 時的檢查清單
- [ ] 顏色來自 `AppColors`？（禁止 hardcode hex）
- [ ] 字重只用 w500/600/800/900？
- [ ] 圓角：按鈕 6、卡片 8？
- [ ] 深底頁面文字都是淺色（白 / white70 / white54）？
