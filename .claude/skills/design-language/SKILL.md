# 今天換誰？設計語言（v3 — 黑底雜誌刊物風）

參考基準：使用者核可的 enter-code mockup「HOUSEHOLD WEEKLY」黑底雜誌版面。

## 核心原則

1. **雜誌刊物感**：像一本每週出刊的家事週刊，不像 SaaS App。
2. **黑底為主**：任務牆頁面底色 #010101，文字白色；卡片是彩色色塊浮在黑底上。
3. **大字刊頭**：標題超大（display 60+）、超粗（w900）。「今天換誰？」的「誰」用粉色強調。
4. **彩色色塊卡**：卡片用實色填滿（藍灰 / 白輪替）、圓角 18、無邊框無陰影。
5. **重點色克制**：粉 #CF729B 是主要 accent（標題誰、獎勵藥丸、進行中、建立鍵、選中狀態）；橘 #FF8B04 用於已完成/次數強調。

## 色票（唯一來源：`lib/theme/app_colors.dart`）

| Token | Hex | 用法 |
|---|---|---|
| ink | #010101 | 任務牆底色、主要按鈕、卡片深色文字 |
| main | #C2D1D3 | 藍灰卡片底 |
| white | #FFFFFF | 白卡片底、深底上的文字 |
| pink | #CF729B | 誰、獎勵藥丸、進行中、建立鍵、選中 tab |
| orange | #FF8B04 | 已完成狀態、次數強調 |
| bg | #F7F7F7 | 淺色頁面（我的任務等尚未黑底化的頁） |

## 元件規則

- **刊頭 Masthead**：`HOUSEHOLD WEEKLY`（白、字距大）+ `NO.07`（粉）→ 粉色分隔線 → 大標（誰=粉）→ 副標「本週待接任務 · N ／ 星期X」(白 70%)。
- **分頁 Tabs**：白字、選中 w800 + 粉色底線；未選中 white38。不用膠囊。
- **任務卡**：藍灰 / 白**輪替**底色、圓角 18、無框無影。版面：左大 emoji、標題大字、發起人小灰字、粉色獎勵藥丸；**次數在右上角**（大數字 + 小 /N）；**動作在右下角**（我要接=黑藥丸按鈕帶 → / 進行中=粉色小字）。
- **獎勵藥丸**：粉底、白字、左側白圓小圖標（🎁 神秘 / ＄ 現金 / 🎀 一般）。神秘未揭曉顯示「神秘禮物」。
- **底部導覽**：白底、圓弧 notch；中央建立鍵 = **粉色圓形**大鍵白色 +；選中項粉色、未選中 inkSoft。iPhone 安全區用 `MediaQuery.padding.bottom`。
- **禁止**：drop shadow、膠囊 tab、通用 SaaS 下拉、超過主色以外的雜色。

## 尚未套用黑底風的頁面（待辦）
任務詳情、發起任務、我的任務、通知、我的 —— 目前仍是淺色。若要全站一致，依本規範逐頁黑底化。

## 修改 UI 時的檢查清單
- [ ] 顏色來自 AppColors？（禁止 hardcode hex）
- [ ] 卡片藍/白輪替、圓角 18、無框無影？
- [ ] 次數右上、動作右下、進行中粉色小字？
- [ ] 重點色克制，黑與白為主？
